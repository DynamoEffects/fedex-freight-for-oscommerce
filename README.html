Introduction

This shipping module is for FedEx's Freight Service. It will contact FedEx's server and give you a quote on all the items in your shipment. It adds additional fields to the product editor in the admin section that are necessary to calculate an accurate price.

IMPORTANT NOTE: FedEx Freight does not support estimates for shipments from one Canadian address to another. Shipping from US to US, Canada to the US, or vice versa works fine.
Requirements

To use this, you need these things (ask your webhost if you don't know):
  • Your server must either have fopen() or cURL enabled and functional

Installation Instructions

Do a complete backup of your store and database before installing this!

STEP #1 - Backup Your Store
  1. Completely backup your store AND database and download them to your computer.

STEP #2 - Upload the new files
  • /catalog/includes/modules/shipping/fxfreight.php
  • /catalog/includes/languages/english/modules/shipping/fxfreight.php

STEP #3 - Update your database
  • Using the included fxfreight.sql file, update your database using phpMyAdmin or similar tool.

STEP #4 - Modify /catalog/admin/categories.php
  • Use the included categories.php file to update your store. Changes can be found by searching for blocks of code between:
    //---FedEx Freight Modification START ---//
    //---FedEx Freight Modification END ---//

    There are a total of 5 required changes.

STEP #5 - Modify /catalog/admin/includes/languages/english/categories.php
  • Use the included categories.php language file to update your store. Changes can be found by searching for blocks of code between:
    //---FedEx Freight Modification START ---//
    //---FedEx Freight Modification END ---//

    There is only 1 required change.

STEP #6 - Enable and Configure Module
  1. Go into your Administration section -> Modules -> Shipping and enable the "FedEx Freight" module.
  2. Click "Edit" and configure the module for your store.
  3. Click "Update" to save your settings.

STEP #7 - Test your installation
  • Add an item to your cart and go to the checkout_shipping.php page to test if FedEx Freight returns a shipping price.
Updating Instructions

Follow these instructions to the letter or you might leave old values in your database!

1. Go into your administrative section -> Modules -> Shipping and copy down all of the settings for this module.

2. Click on "Remove" and uninstall this module.

3. Upload the latest version of this shipping module to includes/modules/shipping/fxfreight.php, overwriting the old one.

4. Click on the "Install" button and reenter all of your settings.
Troubleshooting

Q: FedEx Freight always returns an error!

A: Make sure your products' freight settings are valid and also ensure that your web host allows outward bound connections using fopen or cURL.
